Detailed Description

Used to warm start Distance. Caches particular information from a simplex - a related metric and up-to 3 index pairs.

Invariant
As the metric and list of index pairs should be values from a snapshot of a simplex, the metric and list of index pairs must not vary independent of each other. As such, this data structure only allows these values to be changed in unison via object construction or object assignment.
Note
This data structure is 12-bytes large.
