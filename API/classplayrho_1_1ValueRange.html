Detailed Description

template<typename T>
class playrho::ValueRange< T >

Value range template type.

This type encapsulates a min-max value range relationship.

Invariant
The min and max values can only be the result of std::minmax(a, b) or the special values of positive and negative infinity respectively indicating the "unset" value.
