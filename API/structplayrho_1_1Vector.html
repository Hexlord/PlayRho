Detailed Description

template<std::size_t N, typename T>
struct playrho::Vector< N, T >

Vector.

Basically a constexpr and constructor enhanced std::array for C++14.

Note
This type is trivially default constructible - i.e. default construction performs no actions (no initialization).
This type should be drop-in replacable with std::array in C++17.
Examples:
HelloWorld.cpp.
