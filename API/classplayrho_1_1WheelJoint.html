Detailed Description

Wheel joint.

This joint provides two degrees of freedom: translation along an axis fixed in bodyA and rotation in the plane. In other words, it is a point to line constraint with a rotational motor and a linear spring/damper.

Note
This joint is designed for vehicle suspensions.
