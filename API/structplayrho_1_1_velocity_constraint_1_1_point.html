Detailed Description

Note
Default initialized to values that make this point ineffective if it got processed counter to being a valid point per the point count property. This allows both points to be unconditionally processed which may be faster if it'd otherwise cause branch-misprediction and depending on the cost of branch mis-prediction.
This structure is at least 36-bytes large.
