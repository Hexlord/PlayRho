Detailed Description

template<typename BASE_TYPE, unsigned int FRACTION_BITS>
class playrho::Fixed< BASE_TYPE, FRACTION_BITS >

A fixed-point template class.

This is a fixed point type template for a given base type using a given number of fraction bits.

Note
For a 32-bit sized fixed point type with a 14-bit fraction part 0.000061035156250 is the smallest double precision value that can be represented.
